# Cartoon Movie Website

## Project Overview

This project is a front-end website designed for browsing cartoon movies. It was developed as a university project using HTML, CSS, and JavaScript.

The website provides a simple and interactive user interface where users can navigate through different pages using watch buttons .

## Features

- Modern homepage design
- Interactive navigation menu
- Watch buttons
- Hover effects
- Multiple web pages

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Purpose

This project was created for educational purposes to practice front-end web development concepts and improve UI design skills.

## Note

This is a front-end prototype only. It does not include a backend, database, or actual video streaming functionality.
